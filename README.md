# Решения тестовых заданий

Задания выполнены отдельными страницами внутри папки `docs`. Общая страница со ссылками находится в `docs/index.html`.

## Текст

Тут просто написал немного про себя и использовал некоторое число тегов.

## Форма

В CSS использовал переменные:

- `--label-width`
- `--field-gap`
- `--control-width`

Так удобнее держать выравнивание формы в одном месте. Если нужно изменить ширину подписей или полей, не придется править каждую строку отдельно.

## Кнопки

Сделал стилизацию через переменные CSS. Чтобы быстрее добавлять новые виды.

## Схема

Это задание я не стал верстать, потому что в нем основная сложность не в HTML/CSS, а в ручном переносе координат из картинки.

Если бы делал, выбрал бы такой подход: контейнер размером с макет, карточки людей обычными HTML-блоками с `position: absolute`, а связи отдельным SVG-слоем под ними.

## Витрина

Через CSS стилизовал слайдер, чтобы он был ближе к макету.

Изображения взяты из приложенного `slider/slider.png` и используются как CSS-спрайт: каждая карточка показывает нужный участок общей картинки через `background-position`.
